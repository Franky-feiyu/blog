# 简介：
Markdown是一种轻量级的标记语言，它能让我们使用易读易写的纯文本格式编写文档，然后转换成有效的HTML文档。常用于博客或邮件的编写。
它能让写作者专注于文章的内容，不用在编辑时去考虑杂七杂八的样式。而把样式交给不同平台自己的css解决。由于遵循同一markdown规范，各平台展示文章时都能有一个比较趋同的格式跟自己独有的样式风格。
